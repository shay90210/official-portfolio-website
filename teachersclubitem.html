The Teachers' Club Bootstrap Website

E-commerce website that sells teacher apparel.

View Live Site View Code

Project Goals

  • To recreate an ecommerce website for teachers using HTML, CSS, JavaScript & Bootstrap.
  • To create an efficient website that is accessible to all users.
  • To practice utilizing the UI framework Bootstrap to create a responsive design.

Included Bootstrap Components

  • Nav & Navbar
  • Carousel
  • Container, Row, & Col
  • Accordion
  • Form

JavaScript Features & Functions

  • Generate Cart Items
  • Item Price Calculation
  • Local Storage
  • Basket Storage
  • Increment & Decrement
  • Remove Items from Cart
  • Clear Items from Cart
  • Total Price Amount
