The Teachers' Club Website

Reimagined e-commerce website that sells teacher apparel.

Why an e-commerce website?

The site - www.teachersclubaccess.com is owned by a teacher friend of mine named Jody Reaves who created this site to sell motivating apparel exclusively for teachers. I used this website to practice my skills in SASS/CSS, JavaScript, and BootStrap. This website is based on my own reimagined vision of what her site would look like.

About Page

The About Page was non-existent in the original site. I wanted to make the mission and values of selling the items noticeable to the buyers so that they can see the value and the individual behind the product. This page consists of an accordion, navbar, footer, and images.


Shop Page

The shop page was created primarily using JavaScript and minor CSS components. The page is extremely interactive to add items to a cart, and the user can see what they have in their shopping cart and remove items as needed.


Contact Page

The Contact Page was kept the same. I used BootStrap methods and the form element to create the entire form. To keep it organized, I used the card class and form-control to align all the labels and boxes.


What's next for this project?

Later on, this project will become an application for my final project in my React course. I created this project to become an official portfolio piece for my professional portfolio. This project is supported by the material that I learned from my bootcamp - Nucamp.
