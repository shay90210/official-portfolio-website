The Teachers' Club Bootstrap Website

E-commerce website that sells teacher apparel.

View Live Site View Code

Project Goals

The site - www.teachersclubaccess.com is owned by a teacher friend of mine named Jody Reaves who created this site to sell motivating apparel exclusively for teachers. I used this website to practice my skills in SASS/CSS, JavaScript, and BootStrap by replicating and recreating her website.

About Page

The About Page was non-existent in the original site. I added the mission and values of selling the items to make it more noticeable to the buyers so that they can see the value and the individual behind the product. This page consists of an accordion, navbar, footer, and images.

Shop Page

The shop page was created primarily using JavaScript to make the cart feature as interactive with the items on the page. This page was challenging as I had to understand how each part of the page would function with the user. For example, I had to render all the items on the page in a group, create reactive increments and decrements, store the items in the local storage of the server to conserve my items in the basket, and multiply the prices of my items in order to receive the total amount. While this portion of the project was challenging, it was rewarding to see it work. I wish the styling portion of this page was better. My images were a bit distorted, however, I was pleased the functionality of it.

Contact Page

The Contact Page is similar to the original website. I used BootStrap methods and the form element to create the entire form. To keep it organized, I used the card class and form-control to align all the labels and input boxes.
